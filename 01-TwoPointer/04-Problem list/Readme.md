# Mastery Checklist

Complete each problem without looking at the solution.

For every problem:

* [ ] Read the constraints.
* [ ] Identify the pointer type.
* [ ] Choose the correct weapon.
* [ ] Dry run on paper.
* [ ] Write the code.
* [ ] Analyze Time Complexity.
* [ ] Analyze Space Complexity.
* [ ] Explain why each pointer moves.

---

# Success Criteria

You have mastered Two Pointers if you can solve these 20 problems **without memorizing solutions** by following your workflow:

1. Read the problem.
2. Identify the pointer type.
3. Select the correct weapon.
4. Derive the pointer movement.
5. Dry run.
6. Code.
7. Explain the logic.

> **These 20 problems cover the vast majority of Array and String Two Pointer concepts asked in FAANG-style interviews.**
